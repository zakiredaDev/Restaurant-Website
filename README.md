Restaurant Website

A modern Arabic restaurant website featuring a responsive design and interactive user interface.

Features

- Responsive design for mobile and desktop devices.
- Interactive food menu.
- Shopping cart functionality.
- Elegant dark theme.
- Built using HTML, CSS, JavaScript, PHP, and MySQL.

Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

Project Structure

- "index.html" - Main website page.
- "CSS/" - Stylesheets.
- "JS/" - JavaScript files.
- "Images/" - Website images and assets.
- "Font/" - Custom fonts.
- "PHP/" - Backend PHP files.

How to Run

1. Clone the repository.
2. Place the project in a local server environment such as XAMPP.
3. Import the MySQL database (if applicable).
4. Start Apache and MySQL.
5. Open the project in your browser.
